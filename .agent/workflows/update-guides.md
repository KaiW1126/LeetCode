// turbo-all

1. 現在アクティブなドキュメント（解いた問題のコード）を読み込む。
2. `LeetCode/AlgorithmGuide.md` を読み込み、既存の内容を把握する。
3. `LeetCode/Blind75/Python/PythonGuide.md` を読み込み、既存の内容を把握する。
4. 解いた問題のコードを分析し、以下を判定する：
   - **アルゴリズム・データ構造の観点**: 使ったアルゴリズムや解法パターンが `AlgorithmGuide.md` にまだ記載されていない、または具体例として追加すべきか。
   - **Python構文の観点**: コード内で使っている Python 構文・関数・テクニックが `PythonGuide.md` にまだ記載されていないか。
5. 追記すべき内容がある場合：
   - アルゴリズム・解法パターン → `LeetCode/AlgorithmGuide.md` の該当セクションに追記する（既存フォーマットに合わせる）。
   - Python 構文・関数 → `LeetCode/Blind75/Python/PythonGuide.md` の該当セクションに追記する（既存フォーマットに合わせる）。
6. 追記すべき内容がない場合はその旨を伝える。
7. 変更した内容を日本語で簡潔にまとめて報告する。
