# 🔢 ビット操作（Bit Manipulation）

## どんなアルゴリズム？

数値を **2進数（0と1の列）** として扱い、ビット単位の演算で高速に処理する手法。
通常の四則演算よりも高速で、特定のパターンでは非常にエレガントな解法になる。

```
例: 10 の2進数表現
  10 = 1010₂

主なビット演算:
  AND (&):  1010 & 1100 = 1000  ← 両方1のビットだけ残る
  OR  (|):  1010 | 1100 = 1110  ← どちらか1なら1
  XOR (^):  1010 ^ 1100 = 0110  ← 異なるビットが1
  NOT (~):  ~1010 = 0101        ← 反転
  左シフト (<<): 1010 << 1 = 10100  ← 2倍
  右シフト (>>): 1010 >> 1 = 0101   ← 半分（切り捨て）
```

**イメージ**: 「電灯のスイッチの列」 — 各スイッチ（ビット）をON/OFFすることで情報を表現する。

## Python での書き方

### 基本演算
```python
# 2進数変換
bin(10)                 # '0b1010'
bin(10).count('1')      # 2（1の数）
format(10, 'b')         # '1010'（接頭辞なし）
format(10, '032b')      # 32ビットに揃える

# 2進数 → 整数
int('1010', 2)          # 10

# ビット演算
a & b     # AND
a | b     # OR
a ^ b     # XOR
~a        # NOT
a << n    # 左シフト（× 2^n）
a >> n    # 右シフト（÷ 2^n）
```

### XOR の特性（超便利）
```python
# XOR の特性:
#   a ^ a = 0   （同じ数のXORは0）
#   a ^ 0 = a   （0とのXORは元の数）
#   交換法則・結合法則が成り立つ

# Single Number: 配列で1つだけの数を見つける
def single_number(nums):
    result = 0
    for num in nums:
        result ^= num  # 重複する数は打ち消し合う
    return result
# [4, 1, 2, 1, 2] → 4^1^2^1^2 = 4^(1^1)^(2^2) = 4^0^0 = 4
```

### ビットシフトでデータを圧縮
```python
# 2つの値を1つの整数に格納
# 上位ビットと下位ビットに分けて保存
value = (upper << 10) | lower     # 格納
upper = value >> 10               # 上位ビットを取得
lower = value & ((1 << 10) - 1)   # 下位ビットを取得
```
